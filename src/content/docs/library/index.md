Внутренний каталог переиспользуемых блоков для статей: стильные мок-интерфейсы,
схемы и анимации, которые вставляются в материалы и слегка настраиваются под
контекст. Раздел служебный — `draft: true`, в прод-сборку не идёт.

## Зачем

Чтобы не плодить скриншоты и однотипную вёрстку: один аккуратный блок —
много применений. Каждый блок параметризуется (пропсы компонента или текст
диаграммы), а не копируется целиком.

## Типы блоков

| Тип      | Технология                 | Пример                |
| -------- | -------------------------- | --------------------- |
| Мок-UI   | Astro-компонент (HTML/CSS) | интерфейс Claude Code |
| Схема    | Mermaid (```mermaid)       | потоки, архитектура   |
| Анимация | Lottie (`lottie-web`)      | декоративное движение |

## Как пользоваться

Открой страницу нужного блока — там живое превью, список параметров и готовый
сниппет для копирования в статью. Компоненты лежат в `src/components/blocks/`.
