## Альтернативные провайдеры (агрегаторы)

Альтернативные провайдеры — это сервисы, которые дают доступ к десяткам AI-моделей через **единый API** и **единый кошелёк**.
Они перепродают доступ к официальным API (Anthropic, OpenAI, Google и др.), но за счёт оптовых скидок и оптимизации
маршрутизации предлагают цены **ниже официальных**.

### Зачем это нужно

- **Один API-ключ для всего:** не надо регистрироваться в Anthropic, OpenAI, Google — достаточно одного агрегатора.
- **Цена ниже официальной:** экономия до 70–80% на одних и тех же моделях.
- **Устойчивость:** если один бекенд лёг — агрегатор сам переключит на другой.
- **Оплата из РФ:** многие агрегаторы принимают российские карты или крипту — проще, чем оплачивать официальные API.

### OpenRouter

[OpenRouter](https://openrouter.ai) — самый известный агрегатор, специализируется на **LLM** (текстовых моделях).

- **300+ моделей:** Claude, GPT, Gemini, Llama, Mistral, DeepSeek, Qwen и десятки опенсорсных.
- **OpenAI-совместимый API:** дроп-ин замена — меняешь `base_url` и ключ, код не трогаешь.
- **Цены:** обычно на 5–30% ниже официальных (зависит от модели и загрузки).
- **Бесплатные модели:** есть модели с нулевой ценой для тестов и прототипов.
- **Рейтинг и отзывы:** по каждой модели видны аптайм, задержка и оценки пользователей.
- **Оплата:** пополнение через Stripe (не все карты РФ проходят), крипта.

Подробнее: [OpenRouter-бекенд для Claude Code](/infra/openrouter-backend/).

### Kie.ai

[Kie.ai](https://kie.ai) — новый агрегатор с упором на **мультимодальность**: изображения, видео, музыка и LLM в одном API.

- **Изображения:** Google Nano Banana, GPT Image 2 — до −86% от официальной цены.
- **Видео:** Google Veo 3.1, Grok Imagine — до −84%.
- **LLM:** Claude Opus 4.7, Sonnet 4.6, GPT-5.5 — до −72%.
- **Музыка:** модели для генерации аудио.
- **Особенность:** не платишь за неудачные генерации (failed tasks = \$0).
- **120 000+ разработчиков, 80+ стран.**
- **API:** совместим с OpenAI-форматом для LLM, для остального — собственный REST.
- **Оплата:** карты, в том числе российские (по отзывам пользователей).

:::tip[Сравнение]
| | **OpenRouter** | **Kie.ai** |
| ------------------------ | --------------------------- | ----------------------------------- |
| Фокус | LLM (текст) | Мультимодальные (видео, картинки, LLM, музыка) |
| Моделей | 300+ | 25+ (растёт) |
| Экономия vs официальные | 5–30% | до 84% |
| OpenAI-совместимый API | Да | Да (для LLM) |
| Бесплатный тир | Да (несколько моделей) | Нет |
| Оплата картами РФ | Частично (Stripe) | Да (по отзывам) |
| Аудитория | 1M+ разработчиков | 120 000+ |
:::

## Официальные провайдеры

### Anthropic (Claude)

- Модели: Claude Opus 4.7, Claude Sonnet 4.6, Claude Haiku.
- Прямой доступ — максимальная скорость и гарантия качества.
- Консоль: [console.anthropic.com](https://console.anthropic.com).
- Требуется верификация телефона, оплата картой (не РФ) или через агрегатор.

### DeepSeek

- Китайский провайдер: DeepSeek-V3, DeepSeek-R1.
- Цены значительно ниже западных аналогов.
- Регистрация: [platform.deepseek.com](https://platform.deepseek.com).
- Принимает карты РФ не всегда — проверять актуальный статус.
