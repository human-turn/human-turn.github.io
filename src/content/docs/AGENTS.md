# Структура контента

Актуальная структура папок в `src/content/docs/`. Обновлять при добавлении/удалении разделов.

| Папка                 | Раздел               | Что внутри                                    |
| --------------------- | -------------------- | --------------------------------------------- |
| `chronology/`         | Хронология           | история AI-инструментов                       |
| `agents/`             | Агенты разработки    | общие сведения о харнесах                     |
| `agents/claude-code/` | Claude Code          | введение, возможности                         |
| `agents/pi/`          | Pi Coding Agent      | введение, конфигурация                        |
| `agents/advanced/`    | Продвинутые техники  | форки, мульти-репо (общее)                    |
| `skills/`             | Скиллы и инструменты | MCP, каталоги скиллов, CLI                    |
| `infra/`              | Инфраструктура       | обзор                                         |
| `infra/litellm/`      | LiteLLM              | прокси, локальные модели                      |
| `infra/providers/`    | Провайдеры           | OpenRouter, DeepSeek, Anthropic               |
| `misc/`               | Прочее               | OpenWebUI и т.д.                              |
| `processes/`          | Практика внедрения   | процессы для больших команд                   |
| `library/`            | Библиотека блоков    | переиспользуемые компоненты (Mermaid, Lottie) |
